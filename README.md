# Employee Data Analysis

A Python project for analyzing employee data using Pandas and NumPy.

## Features

- Data Cleaning
- Data Transformation
- Salary Analysis
- Department-wise Reports
- Employee Filtering
- Date Analysis
- Performance Evaluation

## Technologies

- Python
- Pandas
- NumPy

## Run the Project

```bash
pip install -r requirements.txt
python employee_data_analysis.py
```

## Skills

- Pandas
- NumPy
- GroupBy
- Aggregation
- DateTime Operations
- Data Cleaning
- Querying & Sorting
